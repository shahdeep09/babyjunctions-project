# ANALYSIS.md — Baby Junctions
> Performance tracking rules. Findings feed back into Stage 1 topic decisions.

## THE LOOP
```
Weekly: Instagram + Pinterest + WhatsApp APIs → SM Performance tab
Monthly (1st): GSC + GA4 → Claude analyses → Analytics Feed tab updated
Daily 6AM: Stage 1 reads Analytics Feed → smarter topics generated
```

## 6 PERFORMANCE LENSES

### 1. Keyword Performance
- Which keywords rank 1-10? → Create more on same subcategory
- Which rank 11-20? → Stage 7 improves these
- Which keywords get most clicks? → Prioritise in Stage 1
- New GSC keywords we haven't targeted? → Add to queue

### 2. Blog Type Performance
- Which types rank fastest? → Use more of these
- Which convert to affiliate clicks most? → Weight higher in daily mix
- Which get most social shares? → Use for social-first topics

### 3. Seasonal Patterns
```
Jan: Republic Day sales, winter care
Feb: Valentine's gifts, spring gear
Mar: Holi, FY end sales
Apr: Summer begins, heat rash products
May: Mother's Day, peak summer
Jun: Monsoon begins, waterproof + fungal care
Jul: Indoor toys spike
Aug: Independence Day, Raksha Bandhan
Sep: Navratri prep, Durga Puja gifts
Oct: Diwali, Big Billion Days — BIGGEST MONTH
Nov: Post-Diwali, Children's Day (Nov 14)
Dec: Christmas, winter care, year-end lists
```
Start seasonal content 6 weeks before the event.

### 4. Social Media Performance
| Platform | Track | Feed back |
|---|---|---|
| Instagram | Link clicks, engagement rate, best caption format | Caption structure for Stage 6 |
| Pinterest | Outbound clicks, saves, best subcategory | Pin format + board priority |
| WhatsApp | View rate, link clicks, best time | Broadcast time + format |

**Good thresholds:**
- Instagram: ≥50 link clicks · ≥3% engagement · ≥500 reach
- Pinterest: ≥30 outbound clicks · ≥20 saves · ≥500 impressions
- WhatsApp: ≥40 link clicks · ≥30% view rate

### 5. Product Performance
- Which price ranges convert best? (most Indian parents: ₹500–₹2500)
- Which brands get highest CTR?
- Which products go OOS often? → Avoid featuring unreliable stock
- Products with 200+ reviews convert 2× vs 100-200 reviews

### 6. Content Quality
- High time-on-page → good content → do more of this type
- High bounce rate → mismatch between title and content
- Articles 2500+ words: 40% better time-on-page
- Articles with comparison tables: 25% lower bounce rate

## WEEKLY SM REPORT (every Sunday)
Reads SM Performance tab last 7 days → identifies top 3 posts per platform → writes to Weekly SM Report tab + saves .docx to Drive.

Questions answered:
- Best Instagram post this week + why?
- Best Pinterest pin this week + why?
- Best WhatsApp broadcast + why?
- What to replicate next week?

## WHAT ANALYSIS NEVER DOES
- Never deletes published articles based on performance
- Never changes a live article's primary keyword
- Never removes in-stock products that pass quality filters
- Never overrides seasonal urgency with past performance data
  (e.g. even if monitors underperformed last month, still cover them in October)
